# webpack-demo

此部分对应管理输出部分（清理 /dist 文件夹）

我们使用npm安装了clean-webpack-plugin插件依赖，使用这个插件我们可以清空dist目录里面原来的内容

> 非常好用、非常有用的一个插件

我们修改webpack.config.js文件，配置了clean-webpack-plugin插件

最后，运行npm run build 命令，查看效果

> 打开index.html，我们可以看到Hello webpack和一个按钮

> 注意看控制台打出的数据
